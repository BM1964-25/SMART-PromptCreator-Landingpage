# SMART PromptCreator Landingpage

Eigenständige Landingpage für die browserbasierte App SMART PromptCreator.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS

## Entwicklung

```bash
npm install
npm run dev
```

Lokale URL:

```text
http://127.0.0.1:8172/SMART-PromptCreator-Landingpage/
```

## Build

```bash
npm run build
```

## GitHub Pages Root-Dateien aktualisieren

```bash
npm run publish:root
```
